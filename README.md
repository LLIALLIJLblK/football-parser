
## Установка и запуск

1. Клонируйте репозиторий:

```
git clone https://github.com/LLIALLIJLblK/football-parser.git
```

2. Перейдите в директорию проекта:
```
cd football-parser
```
3. Установите зависимости:
```
poetry install
```
4. Активируйте виртуальное окружение:
```
poetry shell
```
5. Запустите парсер:
```
python3 main.py
```
